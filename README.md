# Homebrew Tap For myPad

Install myPad with Homebrew:

```bash
brew tap abhuri/tap
brew install --cask mypad
```

Upgrade later:

```bash
brew update
brew upgrade --cask mypad
```

Uninstall:

```bash
brew uninstall --cask mypad
```

Remove local app data too:

```bash
brew uninstall --cask --zap mypad
```

Project repository:

https://github.com/abhuri/myPad
